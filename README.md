# WrapChecker

This is just a script I cooked up for something.


```
usage: WrapChecker [-h] [-l LENGTH] [-e ENCODING] [-t TAB_SIZE] [-v] filename

Checks the length of each line in a document to ensure it is wrapped

positional arguments:
  filename

options:
  -h, --help            show this help message and exit
  -l LENGTH, --length LENGTH
                        Length limit of line (default: 90)
  -e ENCODING, --encoding ENCODING
                        File encoding to use (default: utf-8)
  -t TAB_SIZE, --tab-size TAB_SIZE
                        Size of tab characters (default: 4)
  -v, --version         show program's version number and exit
```
